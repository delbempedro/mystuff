    \subsection{Aplica\c{c}\~oes de Vetores na F\'isica}

        Vetores s\~ao grandezas f\'isicas que possuem m\'agnitude e dire\c{c}\~ao.
        Em f\'isica, os vetores s\~ao utilizados para representar grandezas como:
        \begin{itemize}
            \item For\c{c}a
            \item Velocidade
            \item Acelera\c{c}\~ao
            \item Deslocamento
        \end{itemize}
        entre outras.
